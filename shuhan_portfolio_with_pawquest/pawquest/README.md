# PawQuest 🐾

PawQuest is a fun daily mission app for pet owners.  
It gives users one small pet-related quest each day, lets them complete it with a note or photo, and rewards them with funny pet titles and memory cards.

## Features

- Create a pet profile
- Generate daily pet quests
- Shuffle quests
- Complete a quest with a note or photo
- Earn Paw Points and streaks
- Unlock funny pet badges
- Save memories locally in the browser

## Tech Stack

This project uses only:

- HTML
- CSS
- JavaScript
- LocalStorage

No backend is required.

## How to Run Locally

Open `index.html` directly in your browser.

## How to Publish on GitHub Pages

1. Create a new GitHub repository, for example: `pawquest`
2. Upload these files to the repository:
   - `index.html`
   - `style.css`
   - `script.js`
   - `.nojekyll`
   - `README.md`
3. Go to your repository **Settings**
4. Open **Pages**
5. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save
7. Your site should become available at:

```text
https://your-username.github.io/pawquest/
```

## Project Description for Class

PawQuest is a daily mission app that helps pet owners spend more playful and meaningful time with their pets. Unlike traditional pet apps that mainly focus on feeding schedules, health records, or vet appointments, PawQuest focuses on fun, interaction, and emotional connection. Users complete small daily quests, earn badges, unlock funny pet titles, and save shareable pet memory cards.
